# Module 6 - Hotspot Detection

Input fields:

- latitude
- longitude
- crime type

Algorithms:

- DBSCAN
- KMeans
- Kernel Density Estimation

Outputs:

- hotspot summary table
- Folium/Leaflet heatmap
- cluster marker map

Mapbox and Google Maps can consume the generated latitude/longitude hotspot
outputs through their own frontend SDKs.
